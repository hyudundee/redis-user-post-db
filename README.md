### ERD

```javascript

A numnber [id] is used to mark both user and his/her posts
            \
             \
              \
user ->   user[id]
{               \
  first_name:    \
  second_name:    \
  email:           \
  phone:            \
}                    \
post ->   postofuser[id]
[
  {
    post1: {...}
    post2: {...}
    ...
  }
]
```
