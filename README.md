### Definition of file data schema with proof that it is in BCNF.

Define User's Schema

```javascript
const UserSchema = new mongoose.Schema({
	name: {
		type: String,
		required: true,
	},
	email: {
		type: String,
		required: true,
	},
	password: {
		type: String,
		required: true,
	},
	friends: [
		{
			user: {
				type: Schema.Types.ObjectId,
				ref: "users",
			},
			connectDate: {
				type: Date,
				default: Date.now,
			},
		},
	],
	createDate: {
		type: Date,
		default: Date.now,
	},
});
```

Design of Posts' Schema

```javascript
const PostSchema = new Schema({
	user: {
		type: Schema.Types.ObjectId,
		ref: "users",
	},
	text: {
		type: String,
		required: true,
	},
	name: {
		type: String,
	},
	createDate: {
		type: Date,
		default: Date.now,
	},
});
```
